# SocialConnect - Social Network Platform

A modern social networking platform built with Next.js, TypeScript, Node.js, and Express.js. Users can create posts, connect with friends, send messages, receive notifications, and share content in real time.

## Features

* User Authentication
* User Profiles
* Posts & Media Sharing
* Likes & Comments
* Friend Requests
* Real-Time Notifications
* Private Messaging
* Stories
* Responsive Design

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS
* Axios
* React Icons

### Backend

* Node.js
* Express.js
* JWT Authentication
* bcryptjs
* Multer
* WebSocket

## Installation

```bash
git clone <https://github.com/sakinaowais989/SocialNetwork.git>
cd socialconnect
npm install
npm run dev
```
